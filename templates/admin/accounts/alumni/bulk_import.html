Alumni Bulk Import

Import alumni data from a CSV file. The CSV file should contain the following columns:

  • first_name - First name (required)
  • last_name - Last name (required)
  • email - Email address (required, unique)
  • student_id - Student ID (required, unique)
  • branch - Branch (required) - Options: CSE, ECE, EEE, MECH, CIVIL, IT, OTHER
  • year_of_admission - Year of admission (required)
  • year_of_passout - Year of passout (required)
  • mobile_number - Mobile number (optional)
  • cgpa - CGPA (optional)
  • current_workplace - Current workplace (optional)
  • job_title - Job title (optional)
  • current_location - Current location (optional)
  • linkedin_url - LinkedIn URL (optional)
  • achievements - Achievements (optional)
  • willing_to_mentor - Willing to mentor (optional) - true/false
